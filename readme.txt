A smart content generation plugin that helps WordPress administrators automatically generate high-quality articles.


== Description ==

Content Auto Manager is a powerful WordPress plugin that enables website administrators to automatically generate high-quality articles using AI technology. It's designed with SEO and content clustering in mind, making it ideal for websites that require frequent content updates.

Key features include:
* Intelligent content generation based on AI technology
* Theme management for batch creation and management of article topics
* Flexible content generation rules configuration
* Custom publishing rules and scheduling
* Support for multiple AI APIs (OpenAI, Claude, Gemini, etc.)
* Vector embedding and clustering technology to enhance content relevance
* Automatic image generation for articles

Activation authorization contact WX: qn006699


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/content-auto-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Content Auto Manager screen to configure the plugin
4. Configure API settings, rules management, and publishing settings

== Frequently Asked Questions ==
