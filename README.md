# WordPress插件 - AI SEO 内容自动生成管家

## 📖 产品目录

### [仪表盘 (Dashboard)](#dashboard)
- **功能**: 插件核心功能概览、任务状态监控、系统性能展示
- **用途**: 查看所有生成任务的总体状态、系统运行情况、关键指标统计

### [API设置 (API Settings)](#api-settings)
- **功能**: 配置AI模型提供商的API密钥、端点、模型参数
- **用途**: 管理OpenAI、Claude、Gemini等AI平台的连接信息，测试API连接状态

### [图像API (Image API)](#image-api)
- **功能**: 配置图像生成服务提供商、模型参数、API密钥
- **用途**: 管理图像自动生成功能，支持ModelScope、OpenAI、SiliconFlow等服务

### [关键词工具 (Keyword Tool)](#keyword-tool)
- **功能**: 关键词研究、挖掘、分析工具
- **用途**: 智能分析相关关键词，提供百度搜索建议，支持关键词分类和优化

### [规则管理 (Rule Management)](#rule-management)
- **功能**: 定义内容生成规则、模板、变量配置
- **用途**: 创建和管理内容生成规则，控制生成逻辑、内容类型、生成频率等

### [主题任务 (Topic Jobs)](#topic-jobs)
- **功能**: 管理主题生成任务队列、状态、进度
- **用途**: 查看和操作主题生成任务，支持批量操作、状态管理、进度追踪

### [主题管理 (Topic Management)](#topic-management)
- **功能**: 管理AI生成的主题、关键词、分类信息
- **用途**: 查看已生成的主题列表，进行分类、编辑、批量操作等管理功能

### [文章任务 (Article Tasks)](#article-tasks)
- **功能**: 管理基于主题的文章生成任务
- **用途**: 监控文章生成进度、状态、错误处理，支持暂停、恢复、重试等操作

### [发布规则 (Publish Rules)](#publish-rules)
- **功能**: 配置文章发布设置、SEO参数、发布状态
- **用途**: 定义文章发布规则，包括分类范围、发布状态、SEO优化参数等

### [向量聚类 (Vector Clustering)](#vector-clustering)
- **功能**: AI向量分析、聚类计算、相似性匹配
- **用途**: 使用K-means算法对内容进行聚类分析，提升内容匹配精度和质量

### [文章结构 (Article Structures)](#article-structures)
- **功能**: 管理文章结构模板、内容角度、结构受欢迎度
- **用途**: AI生成标准化文章结构，分类管理内容角度，分析结构效果

### [品牌资料 (Brand Profiles)](#brand-profiles)
- **功能**: 管理品牌信息、参考材料、品牌语调
- **用途**: 存储品牌相关信息，用于在文章中保持一致的品牌调性和信息准确性

### [调试工具 (Debug Tools)](#debug-tools)
- **功能**: 系统调试、日志查看、性能监控
- **用途**: 查看系统日志、调试信息，启用/禁用调试模式，系统性能分析

### [变量说明 (Variable Guide)](#variable-guide)
- **功能**: 变量说明、模板变量使用指南
- **用途**: 详细说明所有可用模板变量，提供用法示例和编辑建议

## 项目概述

一款强大的WordPress插件，用于智能生成和管理SEO优化的高质量内容。通过AI技术驱动，帮助WordPress管理员自动生成文章、管理内容结构、优化SEO表现，并提供全面的内容分析系统。

## 🚀 核心功能 {#core-features}

### 🧠 AI内容生成
- **多API支持**：兼容OpenAI、Claude、Gemini、自定义API等多个AI平台
- **智能文章生成**：基于主题和关键词自动生成高质量文章
- **角色定制**：可自定义AI角色描述，提升生成内容质量
- **多语言支持**：支持中文、英文、日文、韩文等多语言内容生成

### 🔍 智能内容分析
- **向量聚类**：使用K-means算法对内容进行聚类分析
- **相似性匹配**：基于余弦相似度的智能内容匹配
- **内容角度分类**：自动将内容归类到知识科普、实操指导等10个固定角度
- **动态角度管理**：支持自定义内容角度并可重新分配

### 📝 文章结构管理
- **结构模板生成**：AI生成标准化文章结构模板
- **内容角度分类**：支持10种标准内容角度和自定义角度
- **结构受欢迎度分析**：统计不同结构的文章访问表现
- **智能关联**：将生成文章与最适合的结构模板关联

### 🎯 关键词研究工具
- **关键词挖掘**：智能分析和挖掘相关关键词
- **Baidu搜索建议**：集成百度搜索建议API
- **关键词分组**：自动将关键词按主题分组
- **SEO优化建议**：提供基于关键词的优化建议

### 🖼️ 智能图片生成
- **多平台支持**：支持ModelScope、OpenAI、SiliconFlow、Pollinations等图像API
- **自动配图**：为文章自动生成相关配图
- **模型选择**：可选择不同AI模型生成特定风格图片
- **批量处理**：支持批量生成和管理图片

### ⚙️ 灵活配置系统
- **发布规则管理**：自定义文章发布规则和条件
- **分类过滤**：灵活的分类使用范围设置
- **API配置管理**：统一管理多个API提供商配置
- **定时任务**：支持定时生成和发布内容

### 📊 任务管理系统
- **任务队列**：智能任务队列处理系统
- **状态管理**：完整的任务状态追踪（待处理、运行中、已完成、失败等）
- **批量操作**：支持任务的暂停、恢复、重试等批量操作
- **超时处理**：自动检测和处理超时任务

### 🔧 调试与监控
- **调试模式**：详细的调试日志记录
- **实时监控**：任务状态和进度实时监控
- **日志管理**：完整的操作和错误日志系统
- **性能分析**：系统性能监控和优化建议

## 📋 系统要求 {#system-requirements}

- **WordPress**: 5.0 或更高版本
- **PHP**: 7.4 或更高版本
- **MySQL**: 5.6 或更高版本
- **内存限制**: 推荐 256MB 或更高
- **执行时间**: 推荐 300 秒或更高

## 🛠️ 安装步骤 {#installation-steps}

### 方法一：通过WordPress后台
1. 下载插件压缩包
2. 进入WordPress后台 → 插件 → 安装插件
3. 上传插件文件并激活

### 方法二：FTP上传
1. 将插件文件夹上传到WordPress的`/wp-content/plugins/`目录
2. 在WordPress管理后台激活插件

### 方法三：Git克隆（开发者）
```bash
cd /path/to/wordpress/wp-content/plugins/
git clone https://github.com/your-username/content-auto-manager.git
```

## ⚙️ 配置指南 {#configuration-guide}

### 1. API配置
1. 进入后台 → 内容自动生成管家 → API配置
2. 选择API提供商并填写相应的API密钥
3. 测试连接确保配置正确

### 2. 发布规则设置
1. 进入后台 → 内容自动生成管家 → 发布规则
2. 设置文章发布规则和分类范围
3. 配置SEO参数和发布选项

### 3. 图像API设置
1. 进入后台 → 内容自动生成管家 → 图像API设置
2. 选择图像生成提供商并配置API信息
3. 测试图像生成功能

### 4. 创建主题任务
1. 进入后台 → 内容自动生成管家 → 主题管理
2. 添加或批量导入主题关键词
3. 配置任务参数并启动生成

## 🔧 故障排除 {#troubleshooting}

### 常见问题
1. **API连接失败**：检查API密钥是否正确，网络连接是否正常
2. **生成内容质量差**：调整提示词模板或更换API模型
3. **任务队列卡住**：检查服务器资源和执行时间限制
4. **图片生成失败**：验证图像API配置是否正确

### 性能优化
- 确保服务器有足够的内存和执行时间
- 定期清理不需要的日志文件
- 使用缓存机制提升访问速度
- 优化数据库性能
