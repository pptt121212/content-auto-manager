## 项目概述

**AI SEO 内容自动生成管家**是一个功能强大的WordPress插件，基于AI大语言模型技术，为内容创作者、SEO优化人员、网站运营团队提供全自动化的内容生产解决方案。插件通过规则化配置、任务队列调度、向量智能匹配等技术，实现从主题生成到文章发布的全流程自动化，大幅提升内容生产效率，降低运营成本。

### 适用场景

- 🎯 **SEO内容站**：快速覆盖大量长尾关键词，建立内容护城河
- 📰 **新闻媒体**：每日批量产出资讯文章，保持内容更新频率
- 🛍️ **电商平台**：批量生成产品描述、评测、购买指南
- 📚 **知识库/文档**：自动化生成帮助文档、FAQ、教程
- 🌐 **多语言站点**：一键生成多语言版本内容
- 🏢 **内容矩阵**：同时管理多个品牌/网站的内容生产

### 核心优势

✅ **全自动化流水线**：从主题生成 → 文章创作 → 自动配图 → 定时发布，无需人工干预  
✅ **多API支持**：集成OpenAI、Claude、Gemini、通义千问、文心一言等主流AI服务  
✅ **智能向量聚类**：基于语义向量自动分类、去重、推荐相似内容  
✅ **品牌调性管理**：多品牌资料配置，确保内容符合品牌风格  
✅ **灵活规则系统**：支持为不同分类、主题、场景配置差异化生成策略  
✅ **容错与恢复**：任务失败自动重试，API故障自动切换备用服务  
✅ **完整监控体系**：仪表盘实时监控产出、成功率、队列状态

## 核心功能模块

### 🎛️ 1. 仪表盘（Dashboard）
**功能说明**：系统运行状态总览，快速掌握内容生产健康度。

**核心指标**：
- 📊 今日/本周/本月文章产出统计
- ✅ 任务成功率与失败原因分析
- 🔢 主题库存量与利用率
- ⚙️ 任务队列状态与处理速度
- 📈 向量生成进度与分类覆盖率

**业务价值**：
- 每日早会通过仪表盘5分钟完成运营日报
- 快速发现API配额耗尽、队列堵塞等异常
- 数据驱动决策，优化规则配置提升ROI

### 🔌 2. API设置（API Settings）
**功能说明**：管理AI服务接口配置，支持多服务商并存。

**支持的AI服务**：
- 🌍 **国际服务**：OpenAI GPT系列、Anthropic Claude、Google Gemini
- 🇨🇳 **国内服务**：阿里通义千问、百度文心一言、智谱GLM、讯飞星火
- 🎨 **图像服务**：通义万相、Midjourney、Stable Diffusion、DALL-E

**核心能力**：
- ✅ 一键测试连接，验证API可用性
- 💰 成本优化：不同场景使用不同成本的模型
- 🔄 容灾备份：主API故障自动切换备用
- 🌐 多语言支持：不同语言内容使用原生语言模型
- 📊 向量配置：独立配置嵌入模型支持向量聚类

### 🖼️ 3. 图像API（Image API）
**功能说明**：为文章自动配图，提升内容视觉吸引力。

**支持场景**：
- 🎯 特色图片：为每篇文章生成主题相关封面
- 📝 段落配图：在文章指定位置插入说明性图片
- 🎨 风格定制：根据品牌调性配置图片风格

**成本优化**：
- 重要文章使用高质量付费API（DALL-E）
- 普通文章使用免费API（Pollinations）
- 支持异步生成，不阻塞文章发布流程

### 🔍 4. 关键词工具（Keyword Tool）
**功能说明**：挖掘搜索引擎联想词，辅助选题与SEO优化。

**数据源**：
- 🔎 Google Suggest：全球搜索趋势
- 📺 YouTube Suggest：视频内容热词
- 🇨🇳 Baidu Suggest：中文搜索热词
- 🛒 Shopping Suggest：电商购物关键词
- 🦆 DuckDuckGo：隐私搜索趋势

**使用流程**：
1. 输入种子关键词（如"AI工具"）
2. 系统自动拓展出50-200个长尾词
3. 导出关键词列表，批量创建主题
4. 根据搜索量和竞争度排序，优先产出高价值内容

### 📋 5. 规则管理（Rule Management）
**功能说明**：内容生产的指挥中枢，定义"如何生成内容"。

**规则类型**：
- 📝 **主题规则**：批量生成文章标题/提纲
- 📄 **文章规则**：将主题扩展为完整文章

**配置要素**：
- 🎯 适用分类：指定规则覆盖哪些分类
- 🤖 AI模型：选择使用哪个API配置
- 📏 生成数量：每次生成多少主题/文章
- ✍️ 提示词模板：自定义AI生成指令
- 🎨 文章结构：指定大纲模板和内容角度
- 🏷️ 品牌资料：关联品牌调性和语言风格

**业务场景**：
- 🆕 新站启动：创建1条主题规则+1条文章规则，快速产出首批内容
- 🎭 差异化策略：不同分类使用不同规则（高质量vs高产量）
- 🔥 热点响应：预设"突发热点"规则，事件发生时快速启用
- 🌐 多品牌管理：为每个客户品牌配置独立规则

### 📑 6. 主题任务（Topic Tasks）
**功能说明**：批量生成文章主题（标题/提纲），建立主题库。

**执行流程**：
```
创建任务 → 选择规则 → 入队调度 → AI生成 → 解析入库 → 完成统计
```

**核心能力**：
- ⚡ 批量生成：一次任务可生成数百个主题
- 🎲 去重机制：基于向量相似度过滤重复主题
- 🔄 自动恢复：失败任务自动重试，支持断点续传
- 📊 进度追踪：实时查看每个子任务的执行状态

**运营策略**：
- 📅 定时任务：每天凌晨自动生成新主题，保持库存充足
- 🎯 按需生成：热点事件发生时，手动触发紧急任务
- 📈 A/B测试：同一规则生成多批主题，对比质量选优

### 📚 7. 主题管理（Topic Management）
**功能说明**：主题库的可视化管理界面。

**核心操作**：
- 🔍 搜索筛选：按分类、规则、状态查询主题
- ✏️ 手动编辑：修改标题、调整分类、删除低质主题
- ➕ 手动添加：导入外部主题或手工创作主题
- 🏷️ 状态管理：标记"未用""已用""已废弃"等状态
- 🎯 向量匹配：为手动主题自动分配最合适的分类

**质量控制**：
- 人工审核：批量预览主题，删除不符合要求的
- 重复检测：通过向量聚类发现相似主题并合并
- 定期清理：删除过时或长期未使用的主题

### 📝 8. 文章任务（Article Tasks）
**功能说明**：将主题扩展为完整文章，发布到WordPress。

**生成能力**：
- 📖 字数控制：800-5000字可配置
- 🎨 结构化内容：支持H2/H3标题、段落、列表、引用等
- 🖼️ 自动配图：调用图像API生成特色图片
- 🔗 内链优化：自动插入站内相关文章链接
- 🏷️ 标签生成：基于内容自动提取关键词标签

**发布策略**：
- ✅ **立即发布**：内容生成后直接上线
- 📝 **草稿待审**：先保存草稿，人工审核后发布
- ⏰ **定时发布**：按内容日历定时发布，保持更新频率
- 🔄 **覆盖更新**：对老文章重新生成，保留URL不变

**质量保障**：
- 🎯 人工审核：抽检模式，重点审核高价值内容
- 📊 性能监控：记录生成耗时、成功率、错误类型
- ♻️ 失败重试：API超时或错误自动重试，最多5次
- 🔍 相似度检测：避免生成与历史文章过于相似的内容

### ⚙️ 9. 发布规则（Publish Rules）
**功能说明**：定义文章发布时的属性配置。

**配置项**：
- 📌 默认状态：发布/草稿/待审核
- 👤 文章作者：指定作者账号
- 📁 默认分类：未指定时的兜底分类
- 🏷️ 默认标签：自动添加的通用标签
- 🔑 SEO关键词：meta keywords配置
- 🖼️ 配图策略：是否启用自动配图、配图数量
- 🎨 文章模板：指定文章结构模板
- 🎯 品牌资料：关联品牌调性

**高级功能**：
- 🔄 自动内链：根据关键词自动插入站内链接
- 📊 数据埋点：添加GA/百度统计追踪代码
- 🔔 推送通知：文章发布后通知微信/邮件
- 📱 同步发布：同步到公众号、头条号等平台

### 🧩 10. 向量聚类（Vector Clustering）
**功能说明**：基于语义向量进行内容聚类和智能推荐。

**核心技术**：
- 🔢 向量嵌入：使用Embedding API将文本转为向量
- 📊 K-Means聚类：自动将相似主题分组
- 🎯 增量更新：新主题自动分配到最近的簇
- 🔍 相似度搜索：快速找到相似内容

**应用场景**：
- 🎯 **自动分类**：手工主题自动匹配到合适分类
- 🔄 **去重检测**：发现重复或高度相似的主题
- 📚 **相关推荐**：在文章底部推荐相似内容
- 📊 **内容分析**：识别网站的内容主题分布

**使用建议**：
- 💾 积累1000+主题后首次训练聚类
- 🔄 每月重新训练一次，保持聚类精度
- 🎯 每100个向量约1个聚类簇（系统自动推荐）

### 🏗️ 11. 文章结构（Article Structures）
**功能说明**：预定义文章大纲模板，规范内容结构。

**结构要素**：
- 📋 章节列表：H2/H3标题大纲
- 🎯 内容角度：每个章节的写作视角
- 📚 参考文献：引用来源和数据支撑
- 📏 字数分配：各章节字数建议

**应用场景**：
- 📝 **标准化输出**：确保同类文章结构一致
- 🎓 **教程文章**：步骤清晰，便于读者跟随
- 📊 **评测文章**：外观→性能→体验→总结
- 💡 **深度分析**：背景→现状→趋势→建议

**智能优化**：
- 📊 **受欢迎度统计**：追踪哪种结构的文章流量更高
- 🔄 **A/B测试**：同主题使用不同结构对比效果
- 🎯 **优先复用**：自动推荐高转化率的结构模板

### 🏢 12. 品牌资料（Brand Profiles）

**应用场景**：

- 🌐 **本地化适配**：同品牌在不同地区的表达差异
- 🎚️ **产品线分级**：旗舰/标准/入门产品的调性区分
- 🆘 **危机公关**：负面舆情时统一口径快速响应

**匹配机制**：
- 🤖 自动匹配：根据主题向量自动选择相关品牌
- ✋ 手动指定：创建主题时明确指定品牌
- 🎯 优先级规则：同时匹配多个品牌时的选择策略

## 安装与配置

### 系统要求

- ✅ WordPress 5.0 或更高版本
- ✅ PHP 7.4 或更高版本
- ✅ MySQL 5.6 或更高版本
- ✅ 至少 256MB PHP 内存限制
- ✅ 支持 WordPress Cron 或系统 Cron
- ✅ 支持 cURL 或 file_get_contents 进行HTTP请求

### 安装步骤

#### 方法一：通过WordPress后台安装
1. 下载插件ZIP压缩包
2. 登录WordPress管理后台
3. 进入"插件" → "安装插件" → "上传插件"
4. 选择下载的ZIP文件并上传
5. 点击"现在安装"
6. 安装完成后点击"启用插件"

#### 方法二：通过FTP上传
1. 解压下载的ZIP压缩包
2. 通过FTP工具连接网站服务器
3. 将解压后的插件文件夹上传到 `/wp-content/plugins/` 目录
4. 登录WordPress管理后台
5. 进入"插件"页面，找到本插件并点击"启用"

### 首次配置

#### 1. 配置AI API（必须）
1. 进入"内容管家" → "API设置"
2. 点击"添加自定义API"或选择预定义API
3. 填写API密钥和相关配置
4. 点击"测试连接"确认可用
5. 保存配置

**推荐配置**：
- 🆓 **免费方案**：使用国内免费API试用额度
- 💰 **成本优化方案**：主题生成用便宜模型，文章生成用中端模型
- 🚀 **高质量方案**：全流程使用GPT-4等高端模型

#### 2. 创建生成规则
1. 进入"内容管家" → "规则管理"
2. 创建第一条主题生成规则：
   - 规则名称：`每日主题生成`
   - 规则类型：`主题生成`
   - 选择API配置
   - 选择目标分类
   - 设置生成数量：20
3. 创建第一条文章生成规则：
   - 规则名称：`标准文章生成`
   - 规则类型：`文章生成`
   - 选择API配置
   - 配置文章字数：1500-2000字

#### 3. 配置发布规则（可选）
1. 进入"内容管家" → "发布规则"
2. 设置默认发布状态（建议先设为"草稿"）
3. 配置作者、分类、标签等
4. 启用自动配图（需先配置图像API）

#### 4. 执行首次任务
1. 进入"内容管家" → "主题任务"
2. 点击"创建任务"，选择之前创建的主题规则
3. 等待任务完成（通过仪表盘监控进度）
4. 进入"主题管理"查看生成的主题
5. 选择优质主题，创建文章任务
6. 文章生成完成后，在WordPress后台审核和发布

### 高级配置

#### 启用向量聚类
1. 确保API配置中已设置向量模型
2. 积累至少500个主题
3. 进入"内容管家" → "向量聚类"
4. 点击"开始生成/重新校准所有聚类"
5. 等待聚类完成（大量数据需要较长时间）

#### 配置品牌资料
1. 进入"内容管家" → "品牌资料"
2. 添加品牌资料，填写详细信息
3. 在创建规则时关联品牌资料
4. 生成内容时将自动注入品牌调性

#### 设置定时任务
1. 确保WordPress Cron正常运行
2. 创建规则时启用"定时执行"
3. 设置执行时间（如每天凌晨2点）
4. 系统将自动在指定时间创建任务

## 常见问题（FAQ）

### Q1：插件支持哪些AI模型？
**A**：支持主流的OpenAI GPT系列、Anthropic Claude、Google Gemini、阿里通义千问、百度文心一言、智谱GLM等。只要API遵循OpenAI标准接口，都可以配置使用。

### Q2：生成的内容会被搜索引擎判定为AI内容吗？
**A**：目前主流搜索引擎（Google、百度）更关注内容质量而非创作方式。建议：
- 通过品牌资料和提示词优化，生成有独特视角的内容
- 人工审核添加个人见解和案例
- 启用向量去重，避免与网上已有内容高度相似

### Q3：一天可以生成多少篇文章？
**A**：取决于：
- API速度：GPT-3.5约2-3篇/分钟，GPT-4约1篇/分钟
- API配额：按你的套餐限制
- 服务器性能：影响任务调度速度

实际案例：大多数网站每天生成30-100篇，高配置可达200-500篇/天。

### Q4：需要人工审核吗？
**A**：建议初期100%审核，熟悉系统后可以降低到20-30%抽检。高风险内容（医疗、金融、法律）建议保持100%审核。

### Q5：插件会影响网站速度吗？
**A**：不会。内容生成在后台队列执行，不影响前端访问。生成完成后就是普通的WordPress文章，访问速度与手工文章一致。

### Q6：API费用大概多少？
**A**：以通义千问为例：
- qwen-turbo：0.002元/千tokens（主题生成）
- qwen-plus：0.008元/千tokens（文章生成）
- 日产30篇1500字文章，月成本约150-300元
- 相比人工编辑成本（月薪8000元），节省95%+

### Q7：支持自动配图吗？
**A**：支持。可配置图像API（通义万相、DALL-E等），自动为文章生成特色图片或段落配图。免费API（Pollinations）也可以使用。

### Q8：可以同时管理多个网站吗？
**A**：可以。每个网站独立安装插件，通过品牌资料区分不同网站的内容风格。也可以在一个网站使用多套品牌资料管理不同栏目。

### Q9：担心生成内容侵权怎么办？
**A**：建议：
- 不要在提示词中要求AI复制特定文章
- 启用向量相似度检测，过滤与已发布内容高度相似的
- 添加原创性说明和免责声明
- 重要内容通过Copyscape等工具检测

### Q10：任务失败了怎么办？
**A**：系统有自动恢复机制：
- 任务失败后自动重试（最多5次）
- 查看"调试工具"定位失败原因
- 常见原因：API限流（切换备用API）、网络超时（稍后重试）、提示词错误（优化提示词）
